# 📊 A/B Testing Analysis for Checkout Redesign using MySQL & Power BI

## 📌 Overview

This project presents an end-to-end **A/B Testing Analysis** for evaluating the effectiveness of a redesigned checkout page. The analysis compares the performance of two user groups—**Control (Variant A)** and **Treatment (Variant B)**—to determine which version delivers better business outcomes.

Using **MySQL** for data analysis and **Power BI** for visualization, this project demonstrates how data-driven experimentation helps organizations optimize user experience and improve key business metrics such as conversion rate and revenue.

---

## 🎯 Business Objective

The primary objective of this project is to analyze whether the new checkout page (Treatment) performs better than the existing checkout page (Control).

The analysis aims to answer the following business questions:

- Does the redesigned checkout page improve the conversion rate?
- Which variant generates higher revenue?
- How does customer behavior differ across devices?
- How does conversion change over time?
- Should the company deploy the new checkout page to all users?

## 📂 Dataset

The project consists of two datasets.

### 1. Experiment Assignments

Contains details about users participating in the A/B test.

| Column | Description |
|---------|-------------|
| user_id | Unique User ID |
| experiment_id | Experiment Name |
| variant | Control / Treatment |
| assigned_at | Assignment Date & Time |

---

### 2. Experiment Events

Contains user interactions after joining the experiment.

| Column | Description |
|---------|-------------|
| user_id | Unique User ID |
| event_type | User Event |
| timestamp | Event Timestamp |
| value | Purchase Amount |
| device | Mobile/Desktop |

# 📌 Key DAX Measures

```DAX
Total Users =
DISTINCTCOUNT(experiment_assignments[user_id])

Total Purchases =
CALCULATE(
    COUNTROWS(experiment_events),
    experiment_events[event_type]="purchase"
)

Total Revenue =
CALCULATE(
    SUM(experiment_events[value]),
    experiment_events[event_type]="purchase"
)

Conversion Rate =
DIVIDE([Total Purchases],[Total Users],0)

Revenue Per User =
DIVIDE([Total Revenue],[Total Users],0)
```
